Потом надо было закомитить файлы ещё раз проверить бранчи и пути. Потом ещё 2 команды. git checkout -b site2 (т.е. создали новый бранч)


И git push test site2 (закинули в репо test наш бранч)


Не забывай про команды проверки бранчей и алиасов. git branch -a; git remote -v
